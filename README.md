Tetris (Single-file)
=====================

คำอธิบาย
---------
เกม Tetris แบบไฟล์เดียว (tetris.html) — เล่นในเบราว์เซอร์โดยไม่ต้องติดตั้งอะไรเพิ่มเติม

วิธีเล่น (Controls)
-------------------
- Arrow Left / A : เลื่อนบล็อกซ้าย
- Arrow Right / D : เลื่อนบล็อกขวา
- Arrow Down / S : เร่งให้บล็อกลงเร็วขึ้น (soft drop)
- Space / W / Arrow Up : หมุนบล็อก
- P : หยุด/เล่นต่อ (Pause)
- R : รีสตาร์ทเกม
- บนมือถือ: ใช้นิ้วปัดหรือปุ่มสัมผัส (ถ้ามีใน UI)

ตั้งค่าและเก็บคะแนน
-------------------
- คะแนนและระดับ (level) ถูกเก็บใน localStorage ของเบราว์เซอร์
- ปรับความดังเสียงได้จากหน้าตั้งค่าในเกม

โฮสต์/แจกจ่าย
---------------
- เปิดไฟล์ `tetris.html` โดยตรงในเบราว์เซอร์ (double-click)
- แนะนำโฮสต์ด้วย GitHub Pages หรือ itch.io สำหรับแชร์ออนไลน์
  - GitHub Pages: สร้าง repository แล้วอัพโหลดไฟล์ จากนั้นเปิด `Settings -> Pages` เพื่อเปิดการเผยแพร่
  - itch.io: สร้าง project ใหม่ แล้วอัพโหลด `tetris.html` หรือ zip ของโฟลเดอร์

การพัฒนาเพิ่มเติม (Ideas)
---------------------------
- เพิ่มเสียงเพลงพื้นหลัง (looped)
- เพิ่มระบบผู้เล่นหลายคนแบบ local/online
- ปรับ accessibility: ปุ่มคีย์บอร์ดที่ปรับได้, โหมด high-contrast
- เพิ่มการแปลภาษา (i18n)

วิธีมีส่วนร่วม
----------------
- สร้าง Git repository และส่ง pull request เพื่อปรับปรุง
